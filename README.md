# Battleship

Battleship game created to practice JavaScript testing through the Odin Project's curriculum.

[Live Demo](https://skim2264.github.io/Battleship/)

## Getting Started

```
git clone https://github.com/skim2264/Battleship.git
cd Battleship
npm install
npm start
```

## Built with

- JavaScript
- Webpack
- CSS
- HTML
- Jest
