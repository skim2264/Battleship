# Battleship

Battleship game created to practice JavaScript testing through the [Odin Project's](https://www.theodinproject.com/lessons/node-path-javascript-battleship) curriculum.

[Live Demo](https://skim2264.github.io/Battleship/)

![battleship](https://github.com/skim2264/Battleship/assets/72099715/6f8eeaa6-7078-4091-ac91-22cf709ff552)

## Built with

- JavaScript
- Webpack
- HTML/CSS
- Jest

## Getting Started

```
git clone https://github.com/skim2264/Battleship.git
cd Battleship
npm install
npm start
```
## Features
1. **Classic Battleship Gameplay:** Enjoy a faithful recreation of the classic Battleship game with all its strategic elements and gameplay mechanics.
2. **User-Friendly Interface:** Experience a user-friendly interface that enhances the gaming experience and makes it easy for players to interact with the game.
3. **Autoplace and Drag-and-Drop Ships:** Allow players to choose between auto place or drag-and-drop methods for positioning their ships on their gameboard, providing flexibility and convenience.
4. **Play Against the Computer:** Engage in exciting battles against the computer, testing your strategic skills and decision-making abilities.
5. **HTML, CSS, and JavaScript:** Develop the game using the fundamental web technologies of HTML, CSS, and JavaScript, ensuring compatibility across different platforms and browsers.
6. **Webpack Integration:** Utilize Webpack to bundle and manage the game's assets and dependencies, optimizing performance and facilitating development.
7. **Jest Testing:** Implement comprehensive unit tests using Jest to ensure the game functions correctly and as expected, enhancing its reliability and stability.
